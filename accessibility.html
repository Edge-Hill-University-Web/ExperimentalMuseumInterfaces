Accessibility Statement

At Experimental Museum Interfaces (EMI), we are committed to making our website accessible to all users, including those with disabilities. We aim to create an inclusive digital environment that allows everyone to explore and engage with museum collections easily.

Accessibility Standards Compliance

The EMI website complies with recognised web accessibility standards, including:

  • WCAG 2.1 Level AA standards, ensuring a wide range of accessibility features and functionality.
  • Compatibility with assistive technologies, such as screen readers and voice recognition software.

Our site has undergone thorough testing to ensure it meets these standards and provides a seamless experience for all users. We regularly assess and update the website to maintain compliance as accessibility guidelines evolve.

Performance and Accessibility Testing

The EMI website has been rigorously tested using multiple tools to ensure full accessibility:

Accessible Features

  • Keyboard Accessibility: The entire site is navigable using keyboard controls.
  • Alternative Text for Images: All images include descriptive text alternatives to assist users who rely on screen readers.
  • High Contrast Text: We ensure that text and background colours meet contrast requirements for readability.
  • Scalable Content: Text and other elements can be resized without impacting the site's usability.

Ongoing Commitment

Accessibility is an ongoing priority for EMI. We continuously review the site, conduct accessibility testing, and incorporate user feedback to ensure our platform remains usable for everyone.
